⚙️ Conectar con Google Sheets

Sigue estos 4 pasos — solo se hace una vez

1

Crear una Google Sheet nueva

Ve a sheets.new y crea una hoja en blanco. El script creará los encabezados automáticamente.

2

Abrir el editor de Apps Script

En la Sheet: menú Extensiones → Apps Script. Borra el código de ejemplo y pega el código del archivo apps_script_farmatodo.gs que te entregué.

3

Implementar como aplicación web

Clic en Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Acceso: Cualquier persona → Implementar y autoriza.

4

Pegar la URL del Web App aquí abajo

Copia la URL que termina en /exec y pégala en el campo de abajo.
